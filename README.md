### Register 2026/02/04

The API is in its first viable version. It needs polishing and changes that will come up while the frontend is developed.
It has the following resources implemented:

- Authentication, authorization and complete user management.
- Routes for companies, its categories and its resources (workers, equipment)
- Routes for vacations and sick leaves of workers.
- Routes for work sites, rules and schedules of companies on this work sites.
- Routes for time entries to register time control on worksites.
- Routes for contractors and its attendance on worksites.

The client development has not started yet.
